Skills

  • HTML

    • HTML, which stands for Hypertext Markup Language, is the standard markup language used for creating and structuring web pages on the World Wide Web. It provides the basic building blocks and structure of a webpage, defining the content and layout of text, images, links, and other media.

  • CSS

    • CSS, which stands for Cascading Style Sheets, is a style sheet language used to control the presentation and layout of HTML documents. It provides a way to apply styles, such as colors, fonts, margins, padding, and positioning, to HTML elements, enabling web developers to create visually appealing and consistent designs across web pages.

  • JavaScript

    • JavaScript is a high-level, interpreted programming language that allows developers to add interactivity and dynamic functionality to web pages. As one of the core technologies of the web, JavaScript plays a crucial role in creating interactive user experiences, handling data, and managing the behavior of web applications.
